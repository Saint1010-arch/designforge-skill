designforge 使用说明

【这是什么】
designforge 可以分析任意网站的设计语言（配色、字体、布局、阴影圆角等），
并产出一份设计报告 + 一段“同款提示词”，还能直接生成同款 Next.js 网站，
或把两个网站按权重融合成一个新风格的网站。

【怎么用】
1) 双击「启动.bat」。
2) 浏览器会自动打开本地界面（http://localhost:4571）。
3) 点右上角「设置 API Key」，填入你自己的大模型 Key。
   - 仅保存在你本机浏览器，不上传服务器。
   - 支持任何 OpenAI 兼容接口（可自定义 Base URL 和模型名）。
4) 三个功能页签：
   - 分析：输入网址 → 出设计报告 + 同款提示词。
   - 做同款：输入网址（可填写改写要求，如换标题/换主色）→ 生成网站项目。
   - 融合：输入两个网址 + 拖动权重滑块 → 生成融合风格网站。

【生成的项目在哪】
保存在你运行目录下的 designforge-out 文件夹里。
进入对应子文件夹，执行 npm install 然后 npm run dev 即可本地预览。

【说明】
- 提取网站用的是内置的无头浏览器，无需联网下载。
- 只有调用大模型的功能（报告/生成）需要你的 API Key。
- 任何功能都会先给你一份报告再继续。
